Zipcode

Pokemon App

Pokemon App displays data inside server-side rendered views. When a user goes to the pokemon route they will see an index of pokemon: the names of each pokemon rendered to the page. When a user clicks on the name of the pokemon, they will be taken to that pokemon's showpage, and will see the pokemon's name and image.
